HTML

  • Hypertext Markup Languauge (HTML)- made up of elements.
    add contents to a page: Text elements, image elements, header elements, list elements
    divide and describe a page: Meta-related elements, content division elements, semantic elements
  • The head element contains information (metadata) about the webpage and are not visible to the user.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element, never the closing element
  • components of the head: meta ( defines metadata such as the character set (charset), description, keywords, author, and viewpoint), title (title of webpage,seen on tab *SEO*), style (defines CSS styles associated with website), base (defines base URL for a webpage), link ( connects and external resource to the HTML document), script (embeds client-side scripts in the HTML doc)
  • To link: link rel="stylesheet" href="./assets/style.css"
  • !DOCTYPE html declares what document type to browser
  • html lang="en" tells browser the language is english
  • tag: what appears inside the angle brackets
  • h tags represent the level of heading a text block represents
  • p tag represents a paragragh or block of text
  • ul, ol, li: represent unordered lists, ordered lists, and list items. ul-bullets ol-numbers
  • img tag: contains info on images that are displayed on webpage
    src attribute defines location of image
  • a href: a-anchor and href- attribute points to the url for the link
  • section elements: define a section in a doc that containts thematic content
    section class="card" id="html-section" /section
